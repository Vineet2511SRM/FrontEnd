HTML Entities

HTML entities are used to display special characters in HTML. Here are some examples:

  • & - Ampersand: &
  • < - Less than: <
  • > - Greater than: >
  • " - Double quote: "
  • ' - Single quote: '
  • © - Copyright symbol: ©
  • ® - Registered trademark symbol: ®
  • ™ - Trademark symbol: ™

  • This is a ★

  • This is a ♥
